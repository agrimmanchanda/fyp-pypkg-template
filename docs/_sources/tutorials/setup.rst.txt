# Set up project
# --------------

Go to your github and create a new repository:

   name fyp<year>-<imperial_username>
   public to be able to use github pages
   Add a README file
    This will set a main branch. If left empty on the first push the branch will be called master instead.

Create repository

Click on main a create a new branch called gh-pages
   write gh-pages on the search box
   click on create branch from main (basically empty) below

Download this repository as .zip into your local machine
Clone your repository on your local machine

   The link can be found by clicking on the green code button
   https://github.com/<github_username>/<repository_name>.git

Create project folder in your local machine

   mkdir fyp-test
